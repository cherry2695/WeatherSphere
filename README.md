# WeatherSphere
WeatherSphere is a modern weather dashboard that provides real-time weather updates, forecasts, and climate insights.
A smart weather intelligence platform that transforms real-time weather data into meaningful insights. The website is properly integrated with Leaflet.js and Open Weather API to properly provide accurate results

## 🚀 Features

- 🌍 Search weather by city or location
- 🌡️ Real-time temperature and weather conditions
- 📅 Multi-day weather forecast
- 💧 Humidity monitoring
- 🌬️ Wind speed and direction tracking
- 🌅 Sunrise and sunset information

## 🛠️ Tech Stack

- Frontend : React, TypeScript (TSX),Tailwind CSS
- Backend : Node.js Express.js
- Map Integration : Leaflet.js
- Charts & Visualization : Recharts
- Icons : Lucide React
- Notifications : Sonner
- API : Open Weather API Key
